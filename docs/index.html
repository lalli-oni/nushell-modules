This is the webpage for nushell-modules. A project dedicated to creating script modules and guides for nushell, a modern shell.

We intend to create nushell scripts to streamline Developer Experience. These scripts are heavily opinionated in process and integrations to begin with so we can achieve a streamlined user flow. Subsequently we can add more integration and more flows.

Originally I created this script in zsh. Finding the language and environment extremely undfriendly. Making it hard to write good code. Being unsatisfied I looked a bit into other terminal approaches, like ruby or googles zx (js). But adding a language layer on top of the shell breaks my top programming annoyance: mixing programming languages (currently working with tsx, the oriny is not lost on me).

Took surprisingly long to encounter nushell, which seems to tick a lot of my boxes in terms of philosophy. I hope to help address (imo) the biggest flaw of nushell: lack of resources. The documentation is subjectively decent, but I find it lacking in some key aspects which I hope to contribute to either here or directly.
